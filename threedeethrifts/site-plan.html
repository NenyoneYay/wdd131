Overview

Purpose

This website will be a mock storefront to demonstrate knowledge of dynamic web development. The website will display products, allow visitors to build a cart of products and go through a dummy checkout system to "purchase" the items.

Audience

My audience will be people who are looking to purchase 3d printed products, including toys, tools and artwork.

Dynamic elements

Some dynamic elements enacted through Javascript will include a cart that stores Objects with prices, pictures and descriptions. It will also include decision trees with if-else statements for keeping track of certain things such as whether or not sales tax should be factored into price. It will include a search function to filter for products, similar to the Ponder: Recipes project.
